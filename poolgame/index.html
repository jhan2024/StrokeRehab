Pool

A simple game written with JavaScript in HTML5 canvas.

High Scores

Description

I wrote this game in pure JavaScript using the circle collision physics engine outlined in Eric Leong's website here.

The physics for the holes was my own buggy code :)

The goal of the game is to sink every ball without sinking the cue ball in the fewest number of moves. Click to shoot the cue ball; the farther you click from the cue ball the faster it will shoot.
